7700k + z270 ar + 华硕 ROG 1070 O8G 安装黑苹果过程

发表于 2018-03-14 | 更新于 2019-04-10

配置如下:

  • CPU : Intel Core i7-7700K
  • 主板 : ASUS PRIME Z270-AR
  • 内存 : HyperX FURY 2400MHz DDR4 Non-ECC CL15
  • 硬盘 : SSD 960 EVO M.2 250GB
  • 显卡 : ROG STRIX-GTX1070-O8G-GAMING
  • 显示器 : LG 27UD58-B - 27” IPS LED Monitor - 4K UltraHD

参考资料和论坛:

  • Install macOS High Sierra on Any Supported Intel-based PC
  • 远景论坛

前言

本文安装的黑苹果系统为macOS High Sierra 10.13.2（17C88）,在上述配置上亲测可以正常安装使用,各种驱动也可以正常运行，同时后期我还测试了影驰名人堂750ti,不需要修改任何配置也可以完美兼容,如果安装出现问题或者没办法正常运行驱动,请去上述两个论坛中搜索资料或者求助

UniBeast & MutiBeast:分享地址

安装过程

根据tonymacx86推荐的安装过程来进行操作

Step 1: Download macOS High Sierra

一般来说需要一台白苹果来进行这一步的操作,从APP Store中搜索下载macOS High Sierra,安装包会存在/Applications当中

但是据说在Windows上也是可以成功制作启动U盘,请自己查阅资料吧~

STEP 2: Create a Bootable USB Drive with UniBeast

  1. 插入U盘（容量大于16G）

  2. 打开磁盘工具,抹掉U盘,进行格式化,选择Mac OS Extended (Journaled),Mac OS 扩展（日志式）

  3. 下载并打开Unibeast,一直同意并继续下去即可,选择UEFI Boot Mode（需要主板支持）,对于10系以后的显卡,Graphics Configuration不要进行勾选,后续手动安装驱动

  4. 等待时间还是较长的,由于我使用的是type-C接口（双接口U盘）,速度还算快

  5. 将MutiBeast拖入到启动U盘中

STEP 3: Recommended BIOS Settings

  1. 进入BIOS,先加载默认配置

  2. 如果CPU支持VT-d,禁用

  3. 如果系统开启了CFG-Lock,禁用

  4. 如果系统开启了安全系统模式,禁用

  5. 选择启动系统类型为其他

  6. 如果系统开启了IO Serial Port,禁用

  7. 打开支持XHCI Handoff

对了,后续如果要使用Parallels Desktop安装虚拟机,记得安装成功后把VT-d打开,不然会一直闪报错而安装不了虚拟机

STEP 4: Install macOS High Sierra

  1. 开机设置从U盘启动（F12 for Gigabyte motherboards, F8 for ASUS motherboards, F11 for ASrock motherboards）

  2. 选择Boot OS X Install from Install macOS High Sierra

  3. 选择语言,然后打开磁盘工具,将目标磁盘格式化为Mac OS Extended (Journaled),名字为Hackintosh,然后关闭磁盘工具继续安装

  4. 系统将会进行大约两次的重启,选择相同选项启动即可

  5. 安装完成之后,在启动项中选择Boot High Sierra from Hackintosh,进入系统

STEP 5: Post Installation with MultiBeast（安装驱动）

  1. 打开MultiBeast

  2. QuickStart: UEFI Boot Mode

  3. Drivers：

    • Audio：VoodooHDA v2.8.9
    • Disk：3rd Party SATA & 3rd Party eSATA & Intel Generic AHCI SATA
    • Misc：FakeSMC
    • Network：IntelMausiEthernet v2.3.0

  4. Booloaders：Clover UEFI Boot Mode + Emulated NVRAM

  5. Customize:

    • Graphics Configuration：Intel HD 6xx & Intel Graphics Fixup
    • SSDT Options：Sandy Bridge Core i7
    • System Definitions：iMac 14，2

  6. 点击Build 然后 Install

  7. 下载对应显卡驱动 显卡驱动

  8. 打开显卡驱动进行安装

  9. 配置EFI文件:/Volumes/EFI/EFI/CLOVER/config.plist

  10. 重启即可

挂载EFI至硬盘

参考博客

挂载完成后，就可以拔掉U盘，在bios中设置从硬盘启动即可正常启动
