Custom Image Classify with Tensorflow

发表于 2017-05-23 | 更新于 2017-06-03 | 分类于 Machine Learning

use tensorflow inception

开头

之前一直有想过需要做一个图片分类器，答应过一个人，但是一直自己写的网络太不理想，参考了很多文献，太过于复杂的网络也没办法自己实现，于是只能借助谷歌爸爸的实力了，虽然不是自己写的，但是训练完成之后还有一种是自己做出来的东西一样的错觉，主要目的一开始就是为了给我喜欢的花朵分个类，最后就变成所有的图片均可以进行分类了，具体的 inception 网络构造，在网上一查一大把，这里就不再赘述，我们训练的是 inception 网络的最后一层，更改一下输出也会有不错的效果，至于前面就不要太大动作去更改了，第一部分是自己失败的摸索，也放在这里吸取一点经验吧。
