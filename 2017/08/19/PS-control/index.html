PS 基本操作快捷键总结

发表于 2017-08-19 | 分类于 PS

The command of Photoshop

我会把在 PS 当中使用到的所有能用上的快捷键记录下来，希望参考的时候能有用。主要有一些常用的快捷键，同时还有一些可用的字体和颜色供参考，同时还有英文版和中文版之间常见的翻译。在 windows 上将 command 换为 ctrl 就可以了。

工具栏

  • 前景色转换 X
  • 前景色复原 D

图层

  • 复制图层 command + J
  • 合并可见图层 command + shift + E
  • 合并并复制图层 command + shift + option + E
  • 设置图层只与下一层相关联 command + option + G
  • 创建选择图层下方图层 command + New layer
  • 除了该图层全部取消可见 option + eye 图标

图像

  • 充满屏幕 command + 0
  • 自由变换 command + T
  • 去色 command + shift + U
  • 反向选择 command + shift + I
  • 图像放大 command + +
  • 图像缩小 command + -

操作

  • 色阶 command + L
  • 字体间距放大 option + ->
  • 字体间距缩小 option + <-

  • 画笔工具 b

    • 画笔工具属性 F5

  • 移动工具 v

  • 显示参考线 command + H

滤镜

  • 上次滤镜操作 command + F

素材

字体

  • legal tender regular
  • Trajan Pro
  • American Captain Regular
  • Kozuka Gothic Pro Extra Light
  • Cooper Black Regular
  • Alex Murphy Solid Regular
  • DRAFTSMAN
  • Copperplate Gothic Bold

颜色

  • 天空背景 #42BAFF — #0048FF
  • 蓝图线条颜色 #0036A5

翻译

  • screen - 滤色
  • screen - 滤色
  • stroke - 描边
  • multiply - 正片叠底
  • frosted - 磨砂
